RAS-POS system 설치

1. OS상관없이 APM부터 우선적으로 설치되어야 한다.
<APM : Apache, Php, Mysql 구성의 기본 웹서버이다. 다른 것을 사용해도 무방하다.>

2. Mysql에 계정과 테이블을 만들어준다. 계정은 다음과 같다.

	HOSTNAME : LOCALHOST
	ID :	   RASPOS
	PW :	   RASPOSPW
	TABLE :    RASPOS

3. 반드시 동봉된 raspos.sql을 덮어쓴다.

4. Raspberry PI 안에서 사용할 경우 인터넷 익스플로어는 '미도리'를 사용한다.

5. 웹서버의 root directory에 RAS-POS의 코드를 입력한다.

6. Raspberry PI 안이나 설치된 PC에 사용할 경우 인터넷 주소는 http://localhost/이고 외부에서 사용할 경우 http://ip주소/이다.

7. 상품관리, 매상정보를 확인하려면 http://주소/insert.html로 들어간다.
